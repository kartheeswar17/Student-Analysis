This README provides a concise overview of the Student Performance Analysis script. It covers everything from data generation to final visualization.

Student Performance Analysis
This Python script generates a synthetic dataset of 250 students and performs comprehensive data analysis to evaluate academic performance across different departments and demographics.

Core Features
Synthetic Data Generation: Creates realistic student profiles including ID, Name, Department (CSE, IT, ECE), and Year.

Performance Metrics: Generates random scores for Maths, Science, and English, along with Attendance and Internal Marks.

Feature Engineering: * Calculates Total and Average marks.

Assigns Grades (A, B, C, or Fail) based on average scores.

Statistical Analysis: * Provides descriptive statistics (mean, std, min/max).

Identifies top 5 performers and students who failed.

Calculates correlations between attendance and grades.

Visualizations: Produces four key charts:

Bar Chart: Average marks by Department.

Pie Chart: Gender distribution.

Histogram: Distribution of average marks.

Scatter Plot: Relationship between Attendance % and Marks.
